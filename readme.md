# Hi社 #

**还在开发，不能没使用，不过可以协助开发。**

Hi社是个仿[QQ部落](http://buluo.qq.com)的社交网站。

如果你也想弄个社交网站，Hi社或许是你的不二选择！

## 特点欣赏 ##

* 用户可自行创建“社”。
* 可添加话题。
* 支持Markdown。 *不允许嵌入HTML*
* 支持多语言。 *默认语言由后台设定，并非根据用户信息识别。*
