# Hi社 #

Hi社是个仿[QQ部落](http://buluo.qq.com)的社交网站。

如果你也想弄个社交网站，Hi社或许是你的不二选择！

# 特点欣赏 #

* 用户可自己创建区域，即“社”。
* 支持给帖子添加话题。
* 免费、开源。
* 使用基于Markdown语法。
* 支持多语言。 \*默认语言由后台设定，并非根据请求字符串识别。
